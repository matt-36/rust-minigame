# Rust sdl2 minigame

authors: [@matt-36](https://github.com/matt-36) and [@Rob9315](https://github.com/Rob9315)

This project was a test of building games using SDL2 and Rust.

Current features include:

- Character movement
- Sprite animation
- Map generation
- Mini map
- Player health
- Collision
